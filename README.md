# Universal CRM Admin Dashboard

A comprehensive, universal CRM admin dashboard that can be integrated into **ANY website** for complete content management and site editing capabilities.

## 🎯 **What This Is**

**Universal CRM Admin Dashboard** is a powerful, standalone admin interface that plugs into any website to provide:

- **Live Site Editing** - Edit headers, content, and footers independently without affecting other sections
- **Drag & Drop Builder** - Visual content editor with drag-and-drop functionality
- **Media Library** - Complete file management with upload, organization, and optimization
- **Analytics Dashboard** - Visitor tracking, traffic analysis, and behavior insights
- **Theme Manager** - Colors, fonts, and styling customization
- **User Management** - Role-based permissions (Admin/Editor/Viewer)
- **SEO Tools** - Meta tags, keywords, and search optimization
- **Form Builder** - Create contact forms, surveys, and lead capture
- **Plugin Architecture** - Easy integration with any website technology

## 🛠 **Integration Methods**

### **1. Plugin Integration (Recommended)**
```html
<!-- Add to your website's head section -->
<script src="https://cdn.universalcrm.com/v1/universal-cms.js"></script>
<script>
  UniversalCMS.init({
    siteId: 'your-site-id',
    adminUrl: 'https://your-admin.com'
  });
</script>
```

### **2. API Integration**
- REST API endpoints for programmatic control
- Secure authentication with API tokens
- Full CRUD operations for content management

### **3. Widget Integration**

### **4. iFrame Integration**

## 🔧 **Installation & Setup**

### **1. Download & Extract**
```bash
# Extract the downloaded package
unzip universal-crm-admin-complete.zip
cd universal-crm-admin-complete

# Install dependencies
npm install

# Start development server
npm run dev
```

### **2. Configure Integration**
1. Open the admin dashboard at `http://localhost:3000`
2. Choose your integration method (Plugin/API/Widget/iFrame)
3. Enter your website details
4. Follow the setup guide for your chosen method
5. Test the connection

### **3. Customize & Deploy**
```bash
# Build for production
npm run build

# Deploy to your server
npm run deploy
```
